# 機密情報の取り扱いルール

このリポジトリは **public**（誰でも閲覧可能）です。さらに、これは静的サイトなので
`index.html` / `style.css` / `script.js` は**ブラウザにそのまま配信され、訪問者全員がソースを読めます**。

そのため、機密情報の扱いは次の2段構えにしています。

## 1. リポジトリに載せない（`.gitignore`）
`.env`・鍵ファイル・`private/` などは [`.gitignore`](./.gitignore) で除外済みです。
秘密の値は**これらのファイルにだけ書き**、コミットしないでください。

## 2. そもそもクライアントコードに秘密を書かない
静的サイトでは、JavaScript に書いた API キー・パスワードは
**リポジトリが private でも訪問者に丸見え**です。秘密が必要な処理は必ずサーバー側で行います。

- 今の「つぶやきルーレット」は投稿を**各自のブラウザの localStorage に保存**するだけなので、
  サーバーも秘密情報も一切持ちません（漏れる機密がそもそも無い設計）。
- 将来サーバー機能（全員で共有する掲示板など）を足す場合は、
  APIキー等を **Cloudflare Pages の環境変数 / Secrets** や **GitHub Actions Secrets** に保存し、
  コードには書かず参照だけします。

## うっかり公開してしまったら
一度 push した秘密は履歴に残るため、`.gitignore` に足すだけでは消えません。
その値を**無効化（ローテーション）**したうえで、履歴からの削除（`git filter-repo` 等）が必要です。
